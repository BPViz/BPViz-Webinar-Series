Data Visualization Process

Acquire Worksheet - Key Concepts
  • Data sources
    • Open data sources
    • Big Data free data sources
    • Data.gov
    • This is NOT an exhaustive list.

      • Data should be publically available. If you are working with data that is NOT publically available or data that is part of a research project you MUST get consent from ALL parties involved to use the data. If you are working with your own research data, you must get consent from your research advisor before using the data in the workshop. If you have data as part of a collaboration withe others, you MUST get consent to use the data from ALL parties.

      Note: Handling and having access to the data, does not imply ownership of the data. If it is not publically available, you MUST get consent to use it.

  • Data Format
    • Structured vs Unstructured Data
    • Datamation
  • Primary or secondary data
    • Primary and secondary data
  • Data Types (Basic Primitive Types)
    • Character (character, char)
    • Integer (Integer, int, short, long, byte) with a variety of precisions
    • Floating-point number (float, double, real, double precision)
    • Book: Visualzing Data by Ben Fry, chapter 1 (for basic examples)
  • Variables
    • What are the parameters?
    • Dependent variable - variable being tested or examiined
    • Independent vaiable - the variable that is changed or controlled to test the effects on the dependent variable
  • Audience & Assumptions
    • Audience: The demographic of people most likely to be interested in your topic/results/visualizations
    • Assumptions: What aspect of the process are you treating a truth?
Parse Worksheet - Key Concepts
  • List each field and its data type. If you have structured data (excel spreadsheet, for example, list each column heading and the type of data it holds)
  • User defined data types also allowed
  • See Visualizing Data by Ben Fry (Chapter 1, p 8-9)
  • Assumptions: What aspect of the process are you treating a truth?
Mine Worksheet - Key Concepts
  • Basic Descriptors: basic statistical methods that can be used to explore the data.
  • Categorize: Can the data be grouped together? What's similar? What's different?
  • Are the variables:
    • Categorical
      • Nominal, Ordinal, Interval or Ratio
      • Rank
    • Quantitative
      • Discrete, Continuous
    • Qualitatitive
      • Nominal, Ordinal
    Temporal
    • Is the data streaming data?
    • How is it stored (all at one time, over several years, in years, days, minutes,seconds, etc.)
    Range & Distribution
    • What is the distribution of the data?
    • Are there a few values, small size, evenly spread, sparse or dense?
    Forming/Answering Questions: Four Types of Visualizations
    1. Exploratory: Have no hypotheses about the data. Explore data interactively as undirected searches
    2. Confirmatory: Have specific hypotheses about the data. Goal-oriented examination of the hypotheses
    3. Presentation: Facts to be presented are fixed a priori. Select appropriate presentation techniques
    4. Interactive: Interactions with a pre-defined animation
  • Assumptions: What aspect of the Mine stage are you treating as truth?
Filter Worksheet - Key Concepts
Apply
  • Remove duplicate data
  • Remove data unrelated to the question to be answered
  • Provide a description of the filtered data (what remains)
Evaluate
  • Do you have enough data to answer your question(s)?
  • If you do not, what is your next step?
  • Do you have the right data to answer the question?
  • If do not, what is your next step?
Represent - Key Concepts
Create
  • Create two (2) different views (visualizations) using the same data.
  • Charts Explained
  • Choosing a good chart
Critique Worksheet - Key Concepts
  • Its purpose is to highlight both the shortcommings as well as strengths of the visualization(s).
  • Questions:
    • Does the visualization(s) answeer the question to be answered?
    • Is the visualization fit for the purpose/audience?
    • Is the visualization effective (does the vsualization provide the insight expected)?
    • Is the visualization of the right chart type?
    • List one thing that is done well.
    • List one thing that could have been done better.
    • Make some recommendations for refining the visualization.

  • Data Visualization Check List
  • Visual Best Practices (Tableau)
Interact Worksheet - Key Concepts
  • Interactive Features
    • Provide a description of what an interactive version of the visualizations provided in the REFINE worksheet might look like.
    • What features would you make interactive and why?
  • Tools
    • What visualization tools or programs will you use? Why?
  • Data Format
    • Is the data in the correct format for the chosen visualization tool(s) listed in the Tools section above?
    • If no, then what do you need to get the data in the desired format for the tool?
  • Implement
    • Using the tools listed above, create an interactive version of the visualizations described in the REFINE worksheet.
    • If Tableau is used provide the URL to your Tableau workbook.
  • Questions & Assumptions
    1. Does the interactive visualization(s) provide the intended insight? Explain. If NO then revisit the REFINE worksheet.
    2. Does the interactive visualization(s) answer the questions posed in the MINE worksheet? Explain. If NO then revisit the MINE Worksheet. if yes, then proceed to 3.
    3. Are the visualilzations appropriate for the intended audience? Explain.v
    4. What Assumptions are you making?
  • Visual Analysis Best Practices (Tableau)
